```bash
# Start the backend server
python run.py

# Run tests
pytest -v

# Check app imports
python -c "from src.main import app; print('✅ App imports successfully')"

# Reset database
rm pluto.db
alembic upgrade head

# Apply migrations
alembic upgrade head

# View API documentation (when backend running)
# Open: http://localhost:8000/docs

# Open database
sqlite3 pluto.db
```

## 📱 Frontend File Guide

### index.html (Main Template)
- Navigation bar with logo
- Login/register modals
- Home section with hero
- Products section with grid
- Cart section with table
- Checkout form
- Footer

**Key IDs for customization:**
- `navbar` - Navigation bar
- `loginModal` - Login form modal
- `registerModal` - Register form modal
- `homeSection` - Home/hero section
- `productsSection` - Products page
- `cartSection` - Shopping cart
- `checkoutSection` - Checkout form

### api.js (API Client)
**Key Methods:**
- `register(username, email, password)`
- `login(username, password)`
- `listProducts()`
- `addToCart(cartId, productId, quantity)`
- `createCheckout(cartId, address, method)`

### app.js (App Logic)
**Key Functions:**
- `init()` - Initialize app
- `handleLogin()` - Login form handler
- `addToCart()` - Cart add item
- `showSection()` - Navigate sections
- `showAlert()` - Show notification

### main.css (Styles)
**Key CSS Variables:**
- `--primary-color` - Main brand color
- `--secondary-color` - Secondary color
- `--success-color` - Success messages
- `--danger-color` - Error messages

## 💡 Pro Tips

1. **Use DevTools (F12)** - Press F12 in browser, check Console and Network tabs
2. **Check Backend Logs** - Terminal shows errors and API requests
3. **Use Swagger UI** - Open http://localhost:8000/docs to test API
4. **Read Error Messages** - They usually tell you what's wrong
5. **Check Database** - Use `sqlite3 pluto.db` to inspect data
6. **Verify Imports** - Run: `python -c "from src.main import app"`
7. **Test Mobile** - Press Ctrl+Shift+M in DevTools to test mobile
8. **Clear Cache** - Ctrl+Shift+R to hard refresh frontend
